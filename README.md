# 🧠 Página Web de Psicología

## 📖 Descripción

Este proyecto consiste en el desarrollo de un sitio web informativo sobre psicología, diseñado con HTML, CSS y Bootstrap. Su objetivo es ofrecer información sobre la profesión del psicólogo, los servicios disponibles y facilitar el contacto mediante formularios interactivos.

El sitio fue desarrollado como parte de un proyecto académico, aplicando principios de diseño web, accesibilidad, navegación intuitiva y diseño responsivo.

---

## ✨ Características

* Diseño responsivo adaptado a diferentes dispositivos.
* Barra de navegación entre las diferentes páginas.
* Carrusel de imágenes.
* Sección de presentación.
* Información sobre los servicios psicológicos.
* Objetivos del sitio.
* Certificaciones.
* Herramientas utilizadas.
* Formularios de contacto.
* Iconografía mediante Bootstrap Icons.
* Optimización SEO básica (meta etiquetas, favicon y descripciones).

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons

## 🚀 Cómo ejecutar el proyecto

1. Clona este repositorio.
2. Abre la carpeta del proyecto en Visual Studio Code.
3. Ejecuta el archivo `index.html` utilizando Live Server o cualquier navegador web.
